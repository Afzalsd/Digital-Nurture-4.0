Exercise 19 - Customization with Sass

Exercise 19.1 – Setup & Custom SCSS

Terminal Commands
mkdir bootstrap-sass-project
cd bootstrap-sass-project
npm init -y
npm install bootstrap sass
_variables.scss
$primary: #4b0082;
$border-radius: 1rem;
custom.scss
@import "variables";
@import "../node_modules/bootstrap/scss/bootstrap";
Build Sass
npx sass scss/custom.scss dist/custom.css --source-map
